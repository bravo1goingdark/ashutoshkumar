# ashutosh kumar — portfolio

Personal portfolio of **Ashutosh Kumar**, backend & systems engineer.  
Live at **[ashutoshk.pages.dev](https://ashutoshk.pages.dev)**

---

## stack

| Layer | Tech |
|---|---|
| Framework | SvelteKit 2 + Svelte 5 (runes) |
| Styling | Tailwind CSS v4 |
| Writing | mdsvex (`.svx` markdown + Svelte) |
| Deploy | Cloudflare Pages + Workers |
| Fonts | Inter Tight · Instrument Serif · JetBrains Mono |

## dev

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # production build (also regenerates og.png)
npm run preview    # preview the build locally
```

## deploy

```bash
npm run deploy     # build → wrangler pages deploy
```

---

> Built without a template. Every component is hand-written.
